## About the Song Management App

The Song Management App is a full-stack, modern web application for managing a music library. It features a beautiful, responsive React frontend and a Node.js/Express backend with MongoDB Atlas for persistent storage. The app allows users to browse, add, edit, and delete songs, view detailed song information, and enjoy a seamless, animated user experience.

### Key Features
- **Full CRUD**: Add, edit, delete, and view songs with real-time updates.
- **Modern UI**: Responsive, music-themed design with smooth animations and a consistent dark mode.
- **Redux Toolkit + Redux-Saga**: Robust state management and async API handling.
- **API Integration**: Connects to a deployed backend with Swagger/OpenAPI documentation.
- **Mobile-First**: Fully responsive and touch-friendly.
- **Webpack**: Custom manual configuration (no CRA), with environment variable support.
- **Comprehensive Error Handling**: User-friendly error messages and loading states.
- **Not Implemented Pages**: Fun, animated placeholders for unimplemented routes.

## ğŸ”§ Configuration

### API Configuration

The app uses a centralized API configuration in `src/config/api.js`:

```javascript
// Switch between development and production
const API_CONFIG = {
  PRODUCTION: 'https://song-management-backend-9rpz.onrender.com/api',
  DEVELOPMENT: 'http://localhost:4000/api',
  CURRENT: 'PRODUCTION' // Change this to 'DEVELOPMENT' for local development
};
```

**To switch environments:**
1. Edit `src/config/api.js`
2. Change `CURRENT: 'PRODUCTION'` to `CURRENT: 'DEVELOPMENT'`
3. Restart your development server
